Обработка "Получение и возврат материалов".

Обработка предназначена для анализа обеспеченности материалами,  запланированных ремонтов (данные формируются из документов «План-график ППР», «Смета ремонта (заявка на ремонт)»).

В форме выводятся следующие поля для отбора:

·         Подразделение – подразделение-исполнитель ремонта.

·         Склад – склад, по которому выполняется проверка остатков материалов и излишка (дефицита) ТМЦ. Если склад не указан, показываются остатки на всех складах.

·         Период – период, по которому выполняется анализ запланированных ремонтов.

В табличной части формы выводятся следующие данные:

·         Номенклатура – позиция номенклатуры, требуемая для выполнения ремонтов.

·         Ед. изм. – единица измерения номенклатуры.

·         Остаток на складе – количество номенклатуры на складе, указанном отборе верхней части обработки. Если склад не указан, показываются остатки по всем складам.

·         Остаток всего – общее количество номенклатуры по всем складам.

·         Текущий излишек (дефицит) – текущий излишек (со знаком плюс) или недостаток (со знаком минус) требуемой для выполнения ремонтов номенклатуры. Количество номенклатуры рассчитывается из потребностей в материалах по ремонтам, выполнение которых уже начато, и остатка на выбранном складе. После ввода заказа на внутреннее потребление потребность считается обеспеченной.

·         Планируемый излишек (дефицит) – планируемый излишек (со знаком плюс) или недостаток (со знаком минус) требуемой для выполнения ремонтов номенклатуры. Рассчитывается из потребностей в материалах по всем невыполненным ремонтам, остатка на выбранном складе и заказанных документом Заказ на внутреннее потребление материалов.

Пользователю доступны следующие команды:

·         По кнопке «Переместить излишки на другой склад» сформировать документ «Перемещение товаров» для передачи материалов с указанного склада в шапке обработки на другой склад (указывается в формируемом документе).

·         По кнопке «Устранить дефицит заказами» сформировать документы «Заказ на внутреннее потребление» на недостающее количество (покрыть планируемый дефицит). При выборе нескольких позиций номенклатуры (строк материалов), по которым имеется излишек, пользователь может сформировать документ сразу на необходимое количество.

·         При двойном клике по строке с номенклатурой становится доступно отдельное окно с расшифровкой по двум типам данных для выбора:

o    Список ремонтов, где требуется выбранная номенклатура. В формируемом списке отображаются документы плана ремонтов «План-график ППР» и «Смета-ремонта (заявка на ремонт)», из которых возникает потребность в выбранном материале (запчасти). В списке отражается информация о том, введены ли документы «Заказ на внутреннее потребление», на все ли материалы введены документы «Заказ на внутреннее потребление».

o    Список документов «Заказ на внутреннее потребление», в которых присутствует выбранный материал (запчасть)".
