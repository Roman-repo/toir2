Обработка «Общий план работ»

Обработка предназначена для вывода информации в виде диаграммы или таблицы по видам работ или объектам ремонта для подразделения. Устанавливается период отображения и типы оснований, по которым необходимо вывести информацию. Диаграмма и таблица формируется при нажатии кнопки Построить/Обновить. Итогом служит план работ с датами.
