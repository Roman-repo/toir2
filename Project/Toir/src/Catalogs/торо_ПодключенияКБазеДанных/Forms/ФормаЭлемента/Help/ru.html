Справочник «Подключения к базе данных»

 

В форме выводятся следующие поля:

  • Код - код подключения в справочнике.
  • Наименование - название подключения в справочнике.
  • Отключен - при установке данного флага по подключению не будет происходить обмен.
  • Сервер, База данных, Логин и Пароль - параметры подключения к базе данных, в которую записывает данные АСУ ТП.
  • Период получения данных - период, за который получаются данные. С момента прошлого подключения должно пройти больше времени, чем указано в этом поле.
  • Текст запроса чтения - текст запроса, который получает помеченные записи.
  • Использовать пометку данных - при установке этого флага исполняется запрос, указанный в текстовом поле «Текст запроса пометки обрабатываемых данных»
  • Текст запроса пометки обрабатываемых - текст запроса, который каким-либо образом помечает записи для чтения.
  • Использовать удаление данных - при установке этого флага исполняется запрос, указанный в текстовом поле «Текст запроса пометки обрабатываемых данных».
  • Текст запроса удаления обработанных - текст запроса, который либо удаляет помеченные записи, либо помечает их как обработанные.
  • Использовать разбиение данных на порции - записи будут получаться порциям указанного размера (по времени). При установке этого флага в запрос к базе данных добавляются два параметра (дата начала порции и дата окончания порции), эти параметры соответствующим образом должны быть использованы в тексте запроса. Если установлен отступ, то в запрос будет передан параметр со значением «дата начал сеанса – отступ». Если одновременно включено разбиение данных на порции и установлен отступ – дата окончания последней порции будет уменьшена на значение отступа.
  • Размер порции - поле для указания размера порции.
