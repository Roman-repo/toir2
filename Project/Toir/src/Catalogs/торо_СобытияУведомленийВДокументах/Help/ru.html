Справочник «События уведомлений в документах»

В форме выводятся следующие поля:

  • Наименование - название события.
  • Код - код документа
  • Шаблон сообщения - шаблон сообщения при наступлении события.
  • Формировать всплывающие уведомления - признак, определяющий необходимость наличия всплывающего уведомления. 
  • Формировать почтовые уведомления - признак, определяющий необходимость формирования сообщения на почту пользователя.
  • Разделять получателей по подразделениям - если флаг установлен, то уведомление получат только получатели с подразделением, соответствующим документу. Для выявленных дефектов учитывается реквизит "Подразделение-исполнитель". Во всех остальных случаях используется реквизит "Подразделение".
  • N - номер по списку.
  • Пользователь - ФИО пользователя, который будет получать уведомления.
